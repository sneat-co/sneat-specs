# Decisions (ADRs)

Architecture Decision Records — cross-repo decisions with lasting structural
consequences for the Sneat platform. Unlike [`standards/`](../standards/README.md)
(living conventions) and [`spec/`](../spec/README.md) (SpecScore feature specs),
an ADR captures a **point-in-time decision**: its context, what was decided, and
the consequences accepted.

Each ADR is numbered and dated, and carries a status
(`Proposed` / `Accepted` / `Superseded`).
