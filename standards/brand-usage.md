# Brand Usage — "Sneat" never stands alone

**Rule (founder, 2026-07-09):** "Sneat" does not read well on its own, but reads
great with any suffix. In copy, headings, titles and UI text, always pair it:

- **Sneat Co.** — the company / corporate context (sneat.co)
- **Sneat App** — the consumer super-app (sneat.app)
- **Sneat Team** / **sneat.team** — the team-rituals suite
- **Sneat Work** — software delivery
- the **Sneat platform** / the **Sneat ecosystem** / the **Sneat graph** — attributive uses
- **a Sneat product** — attributive; acceptable (the noun is the suffix)

**Avoid:** bare "Sneat" as a standalone noun — "Sneat builds…", "powered by Sneat",
"Sneat's mission". Rewrite to "Sneat Co. builds…", "powered by the Sneat platform",
"the Sneat Co. mission".

Domain-style lowercase forms (sneat.team, sneat.co) are fine in running text where
they name the property itself.

**Scope:** all landings, app UI copy, README intros, investor materials, release
notes. Existing copy converges opportunistically — apply the rule whenever a file
is touched; no dedicated rename sweep required.
